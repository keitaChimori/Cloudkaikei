# 『Cloudkaikei』クラウド会計システム
 
# 『Cloudkaikei』とは？
あなたがお使いの会計ソフトは使いやすいですか？様々な便利機能を使いこなせていますか？  

『Cloudkaikei』は、必要な機能を厳選してシンプルな構成にすることで、ユーザーの利便性向上、会計業務の時間短縮化を目指したシンプルなクラウド会計システムです。  

# 主な機能
>・ユーザー登録、ログイン・ログアウト機能  
>・請求書の作成、編集、削除、出力機能  
>・納品書の作成、編集、削除、出力機能  
>・顧客情報リストの作成、編集、削除、検索機能  
>・売上台帳機能  
>・ユーザー管理（管理者用）
